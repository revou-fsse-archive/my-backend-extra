# Commands History

All in README.

## Create project

```sh
nlx @nestjs/cli new the-app-name
```

## Create a Prisma service

```sh
# ni -g @nestjs/cli
nlx @nestjs/cli generate module prisma
nlx @nestjs/cli generate service prisma
```

## Setup Swagger

```sh
ni @nestjs/swagger swagger-ui-express
```
